# DVP - Data Visualization with Python

This repository contains programming assignments and examples for learning Data Visualization with Python (DVP). It covers basic Python concepts, string manipulation, functions, and various visualization libraries including Matplotlib, Seaborn, Bokeh, and Plotly.

## Structure

The repository is organized by assignment topics:

1. **01_Python_Basics**:
    - Find the best of two test averages.
    - Check for palindromes and count digit occurrences.
2. **02_Functions_and_Conversions**:
    - Fibonacci sequence function.
    - Number system conversions (Binary to Decimal, Octal to Hexadecimal).
3. **03_String_Manipulation**:
    - Analyze sentence (words, digits, uppercase, lowercase).
    - String similarity comparison.
4. **04_Matplotlib_Bar_Scatter**:
    - Bar plots.
    - Scatter plots.
5. **05_Matplotlib_Histogram_Pie**:
    - Histograms.
    - Pie charts.
6. **06_Matplotlib_Linear_Plotting**:
    - Linear plotting basics.
    - Line formatting.
7. **07_Seaborn_Customization**:
    - Customizing plots with Seaborn aesthetic functions.
8. **08_Bokeh_Linegraph**:
    - Bokeh line graphs with annotations and legends.
    - Different types of plots in Bokeh.
9. **09_Plotly_3D**:
    - 3D plotting with Plotly.
10. **10_Plotly_TimeSeries_Maps**:
    - Time series visualization.
    - Map creation with Plotly.

## Prerequisites

To run these examples, you need Python installed. It is recommended to use a virtual environment.

### Setting up Virtual Environment

1. Create a virtual environment:

    ```bash
    python -m venv venv
    ```

2. Activate the virtual environment:
    - **Windows**:

        ```bash
        venv\Scripts\activate
        ```

    - **Mac/Linux**:

        ```bash
        source venv/bin/activate
        ```

3. Install the dependencies:

    ```bash
    pip install -r requirements.txt
    ```

## How to Run

Navigate to the specific directory and run the python file. For example:

```bash
cd 01_Python_Basics
python 01a.py
```
